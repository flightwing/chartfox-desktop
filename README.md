## ğŸ“– About This Project

**ChartFox Desktop** brings the power of the ChartFox web platform to your desktop environment. While ChartFox is an incredible tool for flight simulation, running it in a browser tab isn't always ideal for a dedicated cockpit setup.

This project creates a focused, distraction-free "Electronic Flight Bag" (EFB) experience that sits separately from your browser, making it easier to Alt-Tab or move to a secondary monitor during critical phases of flight.
